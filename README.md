# CRM Financeiro

Sistema de gerenciamento para correspondentes bancários, permitindo o cadastro de estabelecimentos, usuários, funcionários, clientes e consultas financeiras.

Este projeto está sendo desenvolvido como projeto de estudos, simulando um ambiente corporativo com arquitetura em camadas, integração com APIs externas e controle de permissões.

## Tecnologias

Backend

- Node.js
- Express
- TypeScript
- Prisma ORM
- PostgreSQL

Frontend

- React
- Vite
- TypeScript

Outros

- Git
- GitHub
- Postman

## Funcionalidades

- Cadastro de usuários
- Login
- Controle de permissões
- Cadastro de estabelecimentos
- Consulta automática de CNPJ
- Consulta automática de CEP
- Cadastro de funcionários
- Cadastro de clientes
- Dashboard
- Integrações via API
- Webhooks
